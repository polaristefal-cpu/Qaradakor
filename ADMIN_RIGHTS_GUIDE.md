### 3. Страница быстрого поиска (`/admin/user-search`) — только для админов
- ✅ Поиск пользователя по email, имени или телефону
- ✅ Отображение карточки пользователя с информацией
- ✅ Кнопка для выдачи/снятия прав администратора
- ✅ Подсказка с конкретным номером телефона для быстрого доступа

## API Endpoints

### POST `/admin/users/:userId/toggle-admin`
Переключает статус администратора для пользователя.

**Требования:**
- Авторизация: требуется токен администратора
- Метод: POST
- Параметры: `userId` в URL

**Ответ:**
```json
{
  "ok": true,
  "is_admin": true
}
```

**Хранение данных:**
Флаг `is_admin` хранится в KV-хранилище:
- Ключ: `user:{userId}:profile`
- Поле: `is_admin: boolean`

## Использование

### Способ 1: Через таблицу пользователей
1. Перейти в `/admin/users`
2. Найти нужного пользователя (можно использовать поиск или фильтр)
3. Кликнуть на иконку щита в колонке "Действия"
4. Подтвердить действие

### Способ 2: Через быструю форму поиска (рекомендуется)
1. Перейти в `/admin/user-search`
2. Ввести номер телефона: `+77763933336`
3. Нажать "Найти"
4. Кликнуть кнопку "Сделать админом" / "Снять админа"

## Безопасность
- Все операции доступны только администраторам
- Проверка прав выполняется функцией `requireAdmin()` на сервере
- Подтверждение действия перед изменением статуса
