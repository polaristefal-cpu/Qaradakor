# 🎯 БЫСТРАЯ ИНСТРУКЦИЯ — Получить права администратора

## Шаг 1: Войдите в систему
Войдите с вашим аккаунтом (который привязан к +77763933336)

## Шаг 2: Попробуйте Bootstrap
Перейдите на: **`/admin/bootstrap`**

Нажмите кнопку **"Получить права администратора"**

### Что может произойти:

✅ **Успех!** — "Права администратора выданы!" 
→ Вас перенаправит в админ-панель `/admin/dashboard`

❌ **Ошибка: "Admin already exists"** — уже есть администратор в системе
→ Переходите к Шагу 3

## Шаг 3: Если уже есть администратор

1. Перейдите на **`/admin/check`**
2. Увидите список администраторов (email, телефон)
3. Свяжитесь с одним из них
4. Попросите выдать вам права через `/admin/user-search`

---

## ℹ️ Примечание

- Страница `/admin/bootstrap` работает **только один раз** — для первого администратора
- Все последующие администраторы выдаются только существующими админами
- Это мера безопасности

---

## 🆘 Если ничего не работает

Проверьте:
1. Вы действительно вошли в систему? (есть сессия?)
2. Попробуйте обновить страницу
3. Откройте консоль браузера (F12) → вкладка Console → посмотрите ошибки
